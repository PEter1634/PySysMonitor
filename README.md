# PySysMonitor

A lightweight, open-source Python tool to monitor system resources (CPU, RAM, and Disk usage) and log them automatically into a local file.

## Features
- Real-time CPU, RAM, and Disk usage monitoring.
- Automatic logging to a local `system_log.txt` file.
- Clean and human-readable timestamp formatting.
- Lightweight with minimal dependencies.
